Комментарии к тестовому заданию.

Для запуска проекта необходимо иметь предустановленные matplotlib, pandas, numpy, unittest.
Проект поделен на 3 файла:

1) Curves.py - файл с объявлением и реализацией классов, который потом подключается в два других файла
2) Curves_tests.py - файл с юнит-тестами классов SpotCurve, ForwardCurve, DiscountCurve
3) Curves_plot.py - файл, при запуске которого можно наблюдать графики кривых спот, форвард и дисконтная

Каждый из классов поддерживает конструктор от имени входного файла (таких же, как даны в примере. А для ForwardCurve еще необходимо указать дату, 
на которую будут вычисляться значения форвардных ставок). Основные методы каждого из классов - вычислить значения ставок для указанного срока (строкой или числом дней), 
отрисовать график кривой). Все методы имеют говорящие названия. 
Было сделано предположение, что столбец Tenor входного файла может иметь только вид "<число> <МО или YR>".

В решении учтены потенциальные ошибочные ситуации, все обработаны в виде исключений (в т.ч. некорректность входного файла, хотя она не была протестирована, т.к. предлагалось
использовать только данные файлы).
